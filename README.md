# mainlandingpage

Cinematic landing page for **MotusDAO** built with:
- Next.js (App Router + TypeScript)
- Framer Motion (scroll storytelling)
- Three.js (hero ambient scene)
- Tailwind CSS v4

## Design Direction
Clinical Futurism: a fusion of legacy Wix editorial vibes + modern MotusDAO app glassmorphism.

## Local dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm start
```
